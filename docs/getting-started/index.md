# Getting Started

Welcome to BCIpyDummies! This section will help you get up and running.

## Quick Start Path

1. **[Installation](installation.md)** - Set up Python, dependencies, and credentials
2. **[Quickstart](quickstart.md)** - Get running in 5 minutes
3. **[Emotiv Setup](../hardware/emotiv-setup.md)** - Configure your headset

## Prerequisites

Before you begin, ensure you have:

- Windows 10 or 11
- Python 3.9 or higher
- An Emotiv EEG headset
- Emotiv Cortex app installed
- Trained mental commands in EmotivBCI app

## What You'll Learn

By the end of this guide, you'll know how to:

- Install BCIpyDummies and its dependencies
- Connect to your Emotiv headset
- Control a Windows application with mental commands
